# Weather JS app

The weather JS app was built with HTML, CSS and JavaScript! Check my previous repos for the [VUE ShowTheWeather App](https://tgd-weather.netlify.app).

HOW TO USE: Type the city and country in the search box and press Enter. The weather details of the typed location will be displayed.
